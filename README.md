## 🚀 About 🧩

**ThumbGen** is your AI-powered thumbnail wizard, now with a **smart support chatbot via Botpress**!
Turn ideas into **eye-catching thumbnails** in seconds and get instant guidance on usage, accounts, and billing:

* Magic AI prompts for styles & colors 🪄 
* Instant cloud storage ☁️
* Export-ready 1280x720 thumbnails 📐
* Integrated Botpress chatbot 💬for FAQs, prompts, and support

---

## 🖼 Features 🌟🤖

* 🎨 AI-generated thumbnail images

* **Database:** MongoDB 🗄️
* **Cloud Storage:** Cloudinary ☁️
* **AI:** Imagen 🤖
* **Chatbot:** Botpress 💬
* **Deployment:** Vercel 🚀

## ⚡ How to Run Locally 🖥️💨

```bash
# Clone the repo
git clone https://github.com/ishikaprasad22/ThumbGen.git

# Navigate to project
cd ThumbGen

# Install frontend dependencies
cd client
npm install

# Install backend dependencies
cd ../server
npm install

# Run backend server
npm run dev

# Run frontend
cd ../client
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view locally.
